Sentiment Analysis is the process of determining 'tone' within the context of a given comment.

== Description ==

It shows the sentimental value of a comment. The comment may be an attitude, opinion or feeling towards something such as a person, organization, product or location. It also shows the overall sentimental value of a page or a post.
Use our free sentiment analysis API to know the emotion or attitude hidden inside any comment in English.

== Installation ==
1. Upload \"sentiment-analysis\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Request API Key from http://www.sentimentanalysisonline.com/page/api-request/
